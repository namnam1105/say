## say

this project is an `echo` alternative written in `rust` for fun.

## Compilation [Building]

- Install cargo and rustc
- To compile run:
```bash
cargo build --release
```
- To install say run:
```bash
cargo install --path .
```
- Use say


## Syntax

say [OPTIONS] TEXT

Options:
- -f --format: enables formatting with ANSI escapes
- -h --help: print out built-in help
